BS-EYE Agent 설치 가이드 v3.0.2

Step 1: install.bat 실행

  1. install.bat 파일을 관리자 권한으로 실행
    (파일 우클릭 → 관리자 권한으로 실행)
  2. 서버에서 필요한 파일이 자동으로 다운로드됩니다
  3. 20개 설정 항목을 입력합니다 (기본값은 Enter만 누르면 자동 적용)
  4. 입력 완료 후 설정값 요약이 표시됩니다 → Y로 확인
  5. config.ini가 자동 생성됩니다
  6. 시작프로그램이 자동 등록됩니다 (PC 재부팅 시 자동 시작)
install.bat  ← 우클릭 → 관리자 권한으로 실행

Step 2: config.ini 확인/수정

C:\bseye-agent\config.ini를 메모장으로 열어 아래 항목을 확인합니다.

섹션키설명예시
[KIOSK_INFO]device_type장비유형kiosk 또는 server
[SERVER]hosp_cd병원코드 (본사 발급)H088
[SERVER]kiosk_id키오스크 고유번호KIOSK_BS_01
[SERVER]kiosk_name키오스크 이름1층로비
[PRINTER_A4]ipA4 프린터 IP 주소192.168.0.25
[PRINTER_THERMAL]model영수증 프린터 모델HMK-072
[VAN_AGENT]processVAN 프로세스명KocesICPos.exe
[NETWORK]emr_hostEMR 서버 IP192.168.0.100
[NETWORK]emr_portEMR 포트7000

수정 후 저장합니다.

Step 4: Agent 테스트 실행

CMD 또는 PowerShell에서 아래 명령을 실행합니다:

cd C:\bseye-agent
bseye-agent.exe debug
  1. 브라우저에서 http://localhost:8080 접속
  2. 우측 상단 Online (초록색) → 정상
  3. 모든 항목이 초록색 → 정상
  4. 빨간색 또는 회색 항목 → 해당 장비 점검 필요

Step 6: 확인 완료 후

  1. 테스트 실행 중인 Agent 종료: Ctrl+C
  2. PC 재부팅
  3. 재부팅 후 자동 시작 확인: 
    tasklist | findstr bseye
  4. http://localhost:8080 다시 접속하여 정상 동작 확인

서버PC 설치

  1. install.bat을 동일하게 관리자 권한으로 실행
  2. device_type 입력 시 server 입력
  3. 프린터/바코드 체크가 자동으로 비활성화됩니다
  4. 감시 프로세스: bs_number.exe (번호표시시스템)
  5. DB 체크: MSSQL TCP 1433
  6. config.ini에서 [SERVER_MONITOR] 섹션 확인
[SERVER_MONITOR]
process_list = bs_number.exe
db_type = mssql
db_host = localhost
db_port = 1433
db_check = true

자동 업데이트

  • Agent가 60초마다 서버에서 최신 버전을 확인합니다
  • 새 버전이 있으면 자동 다운로드 → 교체 → 재시작
  • 이전 버전은 bseye-agent_old.exe로 백업됩니다

문제 해결

증상조치
Agent 시작 안됨bseye-agent.exe debug로 에러 메시지 확인
포트 8080 충돌netstat -ano | findstr 8080 → taskkill /F /PID [PID]
프린터 감지 안됨ping [프린터IP] 확인, config.ini ip 값 확인
thermal disconnectedconfig.ini model 확인, USB 케이블 재연결
EMR unknown개발환경은 정상. 운영환경은 키오스크 프로그램 실행 후 확인
자동 업데이트 실패bseye-agent_old.exe를 bseye-agent.exe로 이름 변경하여 복구
자동 시작 안됨 CMD에서 수동 등록: 
schtasks /create /tn "BSEYE Agent" /tr "\"C:\bseye-agent\bseye-agent.exe\" debug" /sc onlogon /rl highest /f
