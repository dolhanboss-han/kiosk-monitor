📝 ① 기본정보 입력

키오스크에 설치할 기본 정보를 입력하세요. 저장하면 config.ini 내용이 생성됩니다.
키오스크에 설치된 프로그램 이름을 쉼표로 구분하여 입력하세요. 모르면 비워두고 현장에서 확인 후 수정 가능합니다.

📦 ② 에이전트 설치

아래 "에이전트 설치 시작" 버튼을 클릭하면 설치 파일이 다운로드됩니다.
다운로드된 파일을 "열기"하면 자동으로 설치+실행됩니다.
클릭 → 명령어 복사 → 키오스크 PC PowerShell에 붙여넣기
PowerShell 관리자 모드에서 붙여넣기: 

            

        


        

            
1

            

                
설치 파일 다운로드

                
위 버튼을 클릭하면 bseye-install-키오스크ID.bat 파일이 다운로드됩니다

            

        

        

            
2

            

                
설치 파일 실행

                
브라우저 하단의 다운로드 파일을 "열기" 또는 더블클릭하세요

                
Windows 보안 경고가 나오면 "추가 정보" → "실행"을 클릭하세요

            

        

        

            
3

            

                
자동 설치 진행

                
CMD 창이 열리며 자동으로 진행됩니다:

                
폴더 생성 → 에이전트 다운로드 → 압축해제 → config.ini 생성 → 패키지 설치 → 에이전트 실행
